# Sfeir School Node.JS 200

Commencer par cloner le dépôt:

```bash
git clone git@github.com:Sfeir/sfeir-school-nodejs.git
```

## jour 1: node

Démarrer:

```sh
npm i
```

Lancer les tests:

```sh
npm test -- **/ex01.test.js
```
