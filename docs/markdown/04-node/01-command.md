# La commande node

* Afficher la ligne de commande

```bash
 node --help
```

* Exécuter un script

```bash
 node app.js
```

* Evaluer une expression

```bash
 node --eval "console.log('hello')"
```
```bash
 node --print "1 + 1"
```
